Banking Management System (Console-Based Application)
Developed a console-based Banking System in Java using OOP principles and a menu-driven approach.
Leveraged HashMap for efficient account management and applied file handling for data persistence.
Implemented core functionalities like account creation, deposit, withdrawal, and balance inquiry.

Features:
- Account creation
- Deposit & Withdrawal
- Balance inquiry
- File handling for data storage

Technologies:
- Java
- OOP
- File Handling
- HashMap
